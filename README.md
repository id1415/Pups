# 🤖 Пупс

**Пупс** — Самый дерзкий бот в твоем болоте.
Короче, это не очередной унылый бот-энциклопедия, который будет вежливо лизать тебе пятки. Пупс — это токсичный, ироничный и острый на язык тип, собранный на базе Aiogram 3 и накачанный стероидами от API Airforce и Gemini. Он умеет всё — от генерации картинок до создания музла, но делать это он будет нагло, с подколами и на нормальном дворовом сленге. Если готов терпеть жесткий юмор — читай дальше. Если нет — закрывай вкладку и иди плачь в углу.
---

## 🔥 Что этот зверь умеет?

*   **⚡ Характер реального пацана: Забудь про шаблонные ответы. Пупс спорит, язвит, цинично шутит, кроет матом (если надо) и держит марку.
*   **🎨 Генерация артов: Нарисует тебе хоть черта лысого через модели семейства Flux. Можно задавать кастомный формат (типа 16:9 или 9:16 для мобилок). В данный момент бесплатных генераторов нет, но возможно будут позже, не нойте.
*   **🎵 Личная музыкальная студия: Пупс может сбацать полноценный трек по твоей теме, стилю и тексту или нарезать чистый инструментал.
*   **👁️ Зрение (Vision): Кидай ему фотку. Он оценит, укажет на твои косяки или переделает картинку по твоему капризу. Бесплатных vision нейронок тоже нет.
*   **📈 Угарное Саммари чата: Включаешь функцию, и этот шерлок записывает весь бред, что вы несете в чате, а ровно в 21:00 по Москве выдает эпичный и ржачный пересказ дня.
*   **🤖 Автономный режим: Можно настроить авто-постинг сообщений в выбранные топики каждые полчаса-час, чтобы чат не казался кладбищем.
*   **🔑Личные API-ключи: Не хочешь сидеть на общих лимитах? Привяжи свой ключ от api.airforce через личку и кайфуй на своей квоте.

## 🛠️ Архитектура проекта и структура кода

*   **`bot11_pups.py`:** Мозговой центр. Здесь крутятся роутеры, хендлеры, планировщик задач APScheduler и вся логика ответов.
*   **`bot_dialogue.py`:** Модуль, отвечающий за авто-флуд в топиках, чтобы бот не молчал.
*   **`utils.py`:** Вспомогательный софт — сохраняет память, шифрует ключи юзеров и держит настройки моделей.
*   **`summary.py + gemini.py`:** Твои персональные шпионы. Логируют переписки и скармливают их Gemini для вечернего саммари.
*   **`variables.py`:** Свалка системных настроек, лимитов, токенов и прочих критических переменных.
*   **`get_models_info.py`:** Запрашивает у Airforce свежие списки моделей, чтобы ты не юзал устаревший хлам.
*   **`pupps_info.py`:** Меню справки, которое никто никогда не читает.
*   **`prompt.py`:** Свод правил поведения для Пупса (его золотой стандарт токсичности).
*   **`run.py`:** Кнопка «Запуск». Тыкаешь сюда, и магия погнала.

## 📋 Как им управлять (Команды)

Обращайся к боту строго по имени: пупс

### 🤖 Базовые команды и Настройки
*   `пупс инфо` — Вызвать меню справки (если совсем тугой).
*   `пупс chat` — Посмотреть или сменить текстовую модель общения.
*   `пупс vision` — Выбрать модель, которая будет разглядывать твои фотки.
*   `пупс image, пупс music` — Настроить движки для генерации артов и треков соответственно.

### 🎨 Работа с графикой (не работает в настоящее время)
*   `пупс нарисуй [ваш запрос]` — Родить шедевр по текстовому описанию.
*   `пупс нарисуй [соотношение] [запрос]` — То же самое, но в нужном размере (например, пупс нарисуй 9:16 кибер-шаурмичную).
*   **Редактирование:** Кидаешь фото и пишешь: пупс добавь [деталь] и нарисуй.

### 🎵 Музло
*   `пупс спой [тема]` — Создание трека на заданную тему.
*   **Для продвинутых меломанов:**
    ```text
    пупс спой
    Style: Energetic Synthwave, 80s electronic pop
    Lyrics: В траве сидел кузнечик...
    ```
*   `пупс спой instrumental` — Сделать годный минус без вокала.

### 👑 Команды для Админов чата
*   `пупс 0` — Стереть боту память (если он конкретно заврался или словил шизу).
*   `пупс вкл пересказ` — Запустить логирование чата ради ежедневного саммари в 21:00.
*   `пупс выкл пересказ` — Выключить шпионаж и сжечь логи.

## 🚀 Быстрый запуск для ленивых

### 1. Требования к окружению
Нужен Python 3.10+. Ставим все библиотеки одной командой:

```bash
pip install -r requirements.txt
```

### 2. Получение ключей
Мутим Мастер-Ключ шифрования
Чтобы ключи твоих юзеров не утекли в сеть, их надо шифровать через Fernet. Сгенерируй ключ ОДИН раз через терминал:

```from cryptography.fernet import Fernet

master_key = Fernet.generate_key()
print(master_key.decode())  # Вывод будет выглядеть примерно так: t-8WvV...=
```

Запомни эту строку. Потеряешь её — твои юзеры больше не смогут расшифровать свои API-ключи.

Где брать остальные токены?
1. Telegram Токен: Клянчи у @BotFather. В насторойках включи Allow Groups, отключи Group Privacy и включи Bot-to-Bot communication.
2. Airforce API Key: Забирай бесплатно на api.airforce (нужно будет подтвердить мобилу через Discord/Google).  
3. Gemini API Key: Регистрируйся и забирай на Google AI Studio.
4. ImgBB: Заведи аккаунт на imgbb.com и забери токен в настройках API (скоро эту фичу уберем, так что не привыкай).

### 3. Настройка конфигурации
Создай в корне файл .env и забей его своими явками и паролями:
```env
PUPS_BOT_TOKEN=твой_токен_телеграм_бота
AIRFORCE_API_KEY=твой_api_ключ_airforce
API_KEY_GEMINI=твой_api_ключ_gemini
IMGBB_API_KEY=твой_токен_хостинга_imgbb
MASTER_CRYPTO_KEY=тот_самый_сгенерированный_мастер_ключ_из_шага_2
AIRFORCE_API_URL=https://api.airforce/v1/chat/completions
```

### 4. Погнали!
Запускай скрипт и надейся, что ты ничего не перепутал:
```bash
python run.py
```

⚙️ Middleware (Как устроена слежка)
В проекте пашет HistoryMiddleware. Она делает три важные вещи:  
1. Затыкает ботов: Если в чате начнется базар между ботами, Middleware жестко обрубит им связь, чтобы они не заспамили весь чат. 
2. Пишет логи: Все сообщения аккуратно складываются в базу для вечернего саммари.  
3. Держит контекст: Наполняет скользящую память переписки, чтобы Пупс помнил, кто его только что оскорбил и за что.
